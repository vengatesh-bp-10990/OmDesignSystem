# OM Design System

A Figma plugin that programmatically scaffolds the OM Design System (ZCatalyst-inspired): pages, variable collections, text styles, 383 icons, and atom/molecule components with full variant matrices.

## Quick start
1. Open Figma → create a file named `OM Design System`
2. Plugins → Development → Import plugin from manifest… → select `om-bootstrap-plugin/manifest.json`
3. Run `OM DS Bootstrap → Bootstrap Phase 1.1 (Foundations)` first, then any other `Build X` command
